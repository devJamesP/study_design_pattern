# Singleton Pattern
---

## 특징
- 인스턴스를 유일하게 생성한다.
- 해당 싱글턴 인스턴스의 전역적 접근을 허용한다.

---
### 멀티 스레딩 환경에서...
* 멀티 스레딩 환경에서 동기화가 필요하다.
  1. 정적 객체로 미리 싱글턴 객체를 생성한다.
  2. lazy하게 생성하되 동기화 메소드를 통해 싱글턴 객체를 생성한다.
  3. lazy하게 생성하되 해당 메소드에서 객체 생성부분만 동기화를 한다.
