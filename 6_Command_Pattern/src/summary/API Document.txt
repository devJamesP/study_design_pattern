홈 오토메이션 리모컨 API 디자인
귀사의 리모컨을 위해 다음과 같은 디자인과 API를 구축했습니다. 이 디자인에는 리모컨 코드를 최대한 단순하게 만들어서
가전제품업체로부터 새로운 클래스가 공급되더라도 리모컨 코드 자체는 고치지 않아도 되도록 하는 것에 가장 중점을 두었습니다.
이런 목표를 달성하기 위해서 커맨드 패턴을 도입해서 RemoteControl 클래스와 업체로부터 제공되는 클래스를 논리적으로 분리시켰습니다.
이런 디자인은 리모컨 생산 단가를 낮추는 데는 물론, 차후에 리모컨을 유지보수하는 데 드는 비용을 줄이는 데도 크게 도움이 될 것이라고 믿습니다.
