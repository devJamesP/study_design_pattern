1. 커맨드 패턴에서 고객은 __입니다. : Client(클라이언트)
2. 첫 번째 커맨드 객체에서 제어했던 것 : Light(전등)
3. 웨이트리스도 __라고 할 수 있습니다. : invoker(인보커)
4. 커맨드는 일련의 행동과 리시버를 __해 줍니다. : binds(연결)
5. 인보커와 리시버는 __되어 있습니다. : decoupled(분리)
6. 우리를 다른 회사에 소개시켜준 회사 이름 : Weather-O-Rama
7. 식당에서 파는 야채와 달걀, 햄이 들어가는 요리 이름 : greeneggsandham
8. 우리가 제일 좋아하는 마을 이름 : objectVile(객체마을)
9. 리모컨에서 리시버 역할을 하는 것 : vendorclasses(업체에서 제공한 클래스)
10. 모든 커맨드에 들어있었던 메소드 이름 : execute
11. 요리사 외에 이 사람은 분리되어 있죠 : waitress(웨이트리스)
12. 요청을 수행합니다. : receiver(리시버)
13. 행동과 리시버를 알고 있는 객체 : command
14. 커맨드 패턴으로 이것도 할 수 있죠 : undo
15. 실제 작업 처리 방법을 알고 있는 객체 : receiver(리시버)
16. 웨이트리스는 __는 하지 않습니다. : cook(요리)
17. 커맨드에는 이게 캡슐화되어 있습니다. : arequest(요청)
